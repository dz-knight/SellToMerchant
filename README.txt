SellToMerchant v1.0.5

功能
- 在商店出售可移除卡牌
- 在商店出售可交易遗物
- 在商店出售药水
- 联机模式下支持同步卖卡、卖遗物、卖药水、队友转账
- 进入游戏后自动检查更新，支持自动下载或跳转 GitHub

安装方法
1. 下载对应分支的压缩包：
   - SellToMerchant-stable.zip
   - SellToMerchant-public-beta.zip
2. 解压到：
   Slay the Spire 2\mods\SellToMerchant
3. 最终目录中应包含：
   - SellToMerchant.dll
   - SellToMerchant.json
4. 重新启动游戏

售价规则
- 卡牌/药水
  - Common: 25 金币
  - Uncommon: 37 金币
  - Rare: 75 金币
- 遗物
  - 售价 = 商店原价的一半

限制
- 基础牌、诅咒牌、状态牌、任务牌、事件牌、Token 牌、不可移除牌不能出售
- 起始遗物、事件遗物、Ancient 遗物、不可交易遗物不能出售
- 单次转账上限为 200 金币

自动更新
- 模组进入游戏后会自动检查 GitHub 最新 Release
- 若发现新版本，会弹出更新窗口
- 可选自动下载更新、打开 GitHub 手动下载，或暂不更新
