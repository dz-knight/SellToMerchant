SellToMerchant v1.0.5 使用说明

一、功能简介
- 在商店出售卡牌、遗物、药水换取金币
- 联机模式下支持同步卖卡、卖遗物、卖药水和转账
- 进入游戏后自动检测更新，支持自动下载或跳转 GitHub

二、安装方式
1. 下载与你当前游戏分支对应的压缩包
   - 正式版：SellToMerchant-stable.zip
   - public-beta：SellToMerchant-public-beta.zip
2. 解压到：
   Slay the Spire 2\mods\SellToMerchant
3. 确认目录中包含：
   - SellToMerchant.dll
   - SellToMerchant.json
4. 重启游戏

三、使用说明
- 进入商店后，右侧会出现出售面板
- 每个商店内：
  - 卡牌最多出售 1 次
  - 遗物最多出售 1 次
  - 药水最多出售 1 次
  - 转账最多使用 1 次

四、售价规则
- 卡牌 / 药水
  - Common：25 金币
  - Uncommon：37 金币
  - Rare：75 金币
- 遗物
  - 售价 = 商店原价的一半

五、限制说明
- 基础牌、诅咒牌、状态牌、任务牌、事件牌、Token 牌、不可移除牌不可出售
- 起始遗物、事件遗物、Ancient 遗物、不可交易遗物不可出售
- 单次转账上限 200 金币

六、自动更新
- 进入游戏后会自动检查 GitHub 最新版本
- 若发现新版本，可选择：
  - 自动爬取更新
  - 打开 GitHub 自行下载
  - 暂不更新

七、项目地址
- https://github.com/dz-knight/SellToMerchant
